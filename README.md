## Installation

To fetch root-casue ananlysis script

`curl --output generate_summary.sh https://raw.githubusercontent.com/ljunggren/bz-utils/main/scripts/generate_summary.sh` 

To run

`./generate_summary.sh *.json > root_cause.log
